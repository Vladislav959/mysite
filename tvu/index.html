    • TVU SearchИщите, находите и обрезайте записанные видео и видео в реальном времени намного быстрее с помощью автоматической сортировки по часам отснятого материала. Это поможет найти нужный контент по секундам.
    • TVU MediaMind ApplianceДобавьте в записанные видео и видео в реальном времени автоматические метаданныые, такие как распознавание голоса и лиц. Это позволит выполнить точный поиск по видеоконтенту. Совместимо с любым решением МАМ.
    • TVU Producer 3.0Облачная платформа для производства видео в прямом эфире. Создавайте прямые трансляции с множеством камер за считанные минуты с неограниченным количеством участников, используя TVU Partyline.
    • TVU RPSПроизводите видео в реальном времени удалённо в домашних условиях и REMI по IP с H.265, Full HD, используя до 6 синхронизированных камер.
    • TVU TimelockБеспроводное дистанционное производство видео в прямом эфире. Покадровая синхронизация до 6 видеопередатчиков TVU One.
    • TVU PartylineИнструмент качественной конференц-связи для эффективной удаленной командной работы во время прямой трансляции и виртуальных мероприятий.
    • TVU TalkshowПригласите фанатов присоединиться к вашей прямой трансляцией с помощью камеры своего смартфона. Сделайте вашу трансляцию по настоящему особенной.
    • TVU GridМощное и расширяемое подключение через IP, с маршрутизацией и раздачей видео, содержащее метаданные реального времени.
    • TVU G-LinkРаздача видео в обе стороны по IP, содержащая в себе метаданные реального времени.
    • TVU Producer 3.0Облачная платформа для производства видео в прямом эфире. Создавайте уникальные мультикамерные стриминговые программы и делитесь ими с помощью любых видеоинтерфейсов, включая SDI.
    • TVU Command CenterСледите и контролируйте все продукты TVU с одной централизованной облачной платформы.
    • TVU MediaMind ApplianceАвтоматически добавляйте к своему видео метаданные, такие как распознавание голоса и лиц. Это позволит провести точный поиск по видеоконтенту. Совместимо с любым решением МАМ.
    • TVU ServersМощные сервера для просмотра, управления и обмена всем контентом в реальном времени с одного устройства. Эти устройства входят в экосистему TVU.
